## 🚀 How to Run the Agent

### 1. Prerequisites
*   Python 3.10 or higher.
*   A Gemini API key from [Google AI Studio](https://aistudio.google.com/).

### 2. Quick Start (Automatic Setup)
We provide an executable `run.sh` script that handles virtual environments, updates requirements, and launches the script automatically.

Run it with your Gemini API key:
```bash
GEMINI_API_KEY="your_api_key_here" ./run.sh
```

### 3. Running Specific Projects (Manual Mode)
If you prefer manual control:
```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Export API Key
export GEMINI_API_KEY="your_api_key_here"

# 3. Run on default sample
python main.py

# 4. Run on a specific project plan
python main.py data/project_gamma.json
```

## 📊 Generating the Slide Deck (Phase 3)

The `generate_presentation.py` script automatically reads the outputs of all individual project reports, prompts Gemini to synthesize portfolio-wide insights, and compiles them into a PowerPoint slide deck:

```bash
GEMINI_API_KEY="your_api_key_here" ./generate_presentation.py
```
This generates **`outputs/Monthly_Executive_Report.pptx`**, featuring:
*   Slide 1: **Title Slide** (Executive Portfolio Review)
*   Slide 2: **Portfolio Overview & Health Dashboard**
*   Slide 3: **Core Trends & Insights** (Slippage vs budget Compliance)
*   Slide 4: **Key Risks & Delivery Bottlenecks** (Vendor & Resource limits)
*   Slide 5: **Resource & Financial Optimization**
*   Slide 6: **Strategic Actions & Next Steps**

## 💡 Design Decisions & AI Architecture

1.  **Strict Rule Engine + AI Explanation Pattern**: To avoid hallucinations, the final RAG status is determined by deterministic mathematical rules. The AI model's role is strictly limited to explaining *why* the rules fired and extracting qualitative recommendations.
2.  **Graceful Degredation for Messy Data**: If metrics are omitted or invalid, weights are reallocated dynamically so that available data can still generate a score. The agent includes a **Confidence Score** indicating the percentage of parameters provided.
3.  **Model Selection**: We use `gemini-3.1-flash-lite`, which offers extremely fast response times, fits within the free tier rate limits, and has excellent JSON structured output adherence.
